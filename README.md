# BED_calculator
A programme designed for oncologists to calculate a retreatment Biological Effective Dose (BED) based on an initially applied BED.

For details behind the calculator's operation and usage please see: Changes in the retreatment radiation tolerance of the spinal cord with time after the initial treatment. T. E. Woolley, J. Belmonte-Beitia, G. F. Calvo, J. W. Hopewell, E. A. Gaffney and B. Jones. 2017. International Journal of Radiation Biology.

By using this calculator, you accept that any clinical decision made using the calculator must be the responsibility of the local clinical specialists.
